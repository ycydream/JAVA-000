1.png

-Xms：jvm启动时分配的内存

-Xmx：jvm运行过程中分配的最大内存

-Xss：jvm启动的每个线程分配的内存大小

-Xmn：年轻代的大小

Metaspace：不在堆空间上，而是在本地内存中

DirectMemory：堆外内存

Xss < Xmn < Xms <= Xmx
